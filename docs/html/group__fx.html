jack_default_audio_sample_t FX_Processor::process ( jack_default_audio_sample_t  sample)

Runs the current FX on a sample by sample basis.

This function should be called for each sample in the frame. It will output the processed frame based on what the selected FX is.

void FX_Processor::setParam ( FX_param_types  param,
fxparam  value 
)

Sets a parameter value for any of the FX in the class.

The FX does not have to be active in order to set its parameters. Therefore, default parameters can be set up upon initialization, and they will be remembered as the FX type is switched.

Parameters
paramThe parameter to change as defined in the FX_param_types enum.
valueThe value of the parameter to set.
